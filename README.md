# pygame_2.5d
2.5D game (think Marathon, Wolfenstein, DOOM) in pygame

# Controls
* WASD or arrow keys
* Q/E or mouselook to rotate
* V to switch view
* System quit (CMD-Q or ALT-F4) to exit
